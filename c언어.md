## 포인터

* %u 부호가없는 10진수 0에서 무한대

```c
//a[0] 주소값 10 (번지)
// int는 4바이트
//10번지 14번지 18번지 22번지
int a[] = {14,22,30,38} // a[4]
printf("%u, ", &a[2]); // &a[2]의 주소위치 = 18
printf("%u", a); // 14가 아닌 a의 주소인 10
// a == &a[0]
// 결과 18, 10
```
